## Built for Base

This project is a **Base-native infrastructure tool** designed to index, organize, and expose decentralized applications deployed on the Base network (Ethereum L2).

The directory focuses **exclusively on the Base ecosystem** and is intended to be reused by:
- Base builders
- Ecosystem researchers
- Infrastructure tooling
- Analytics and discovery platforms


---

## Scope

This directory intentionally indexes **decentralized applications deployed on Base**.

---

## What This Is

- A structured registry of Base ecosystem dApps  
- A discovery layer for Base-native applications  
- A foundation for reusable ecosystem data (API, JSON exports, research tooling)
